base辅助类样式

base样式文件包含一些辅助样式类

类 描述 实例
'.pull-left' 左浮动 无
'.pull-right' 右浮动 无
'.center-block' 居中显示 无
'.show' 强制元素显示 无
'.hide' 强制元素隐藏 无
'.close' 关闭按钮(默认右浮动) x
'.caret up' 上拉小三角
'.caret down' 下拉小三角
